view_list
Callback Method
Request Method HTTP GET
Callback URL The customer can specify a callback URL
Example http://USER_CALLBACK_URL?type=dlr&messageid=MESSAGE_ID&to=6588888888&status=0&statuscode=delivered&errcode=1&network=52099
Parameters Definition
Parameters Parameters Definition
type the type of the callback information,'dlr' means delivery receipt
messageId unique ID for successful SMS requests
to destination phone number (international format) network: destination phone number's network code
status basic status of the message:
0 - delivered;
400 - delivery errors
statuscode detailed status of the message, for example:
delivered(0) - the message has been delivered to the destination phone number;
accepted(0) - the carrier accepted to send this message;
failed(400) - the carrier failed to deliver this message;
rejected(400) - the carrier rejected this message;
unknown(400) - unknown status code from the carrier
errcode The customer can specify a callback URL
network destination phone number's network code
